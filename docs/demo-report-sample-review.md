# LIGHT ONE 데모 상담 리포트 샘플 Review

## 1. 문서 목적

이 문서는 LIGHT ONE 서비스에서 촬영 품질은 기본 기준을 충족했지만, 상담 내용상 트레이너가 한 번 더 확인해야 하는 REVIEW 상태를 보여주기 위한 합성 데이터 기반 샘플입니다.

이 문서에는 실제 회원 이미지, 얼굴 이미지, 실명, 연락처, 건강정보, 상담 원본 데이터가 포함되어 있지 않습니다. 모든 내용은 포트폴리오와 심사 설명을 위한 예시입니다.

## 2. 리포트 위치

LIGHT ONE은 PT·피트니스 현장에서 상담 리포트의 신뢰도를 보조하기 위한 비의료 운동상담 보조 SaaS를 목표로 합니다.

본 리포트는 의료 진단, 치료 판단, 재활 처방, 질환 위험도 예측을 제공하지 않습니다. 트레이너가 회원 상담 과정에서 확인해야 할 항목을 정리하는 참고 자료입니다.

## 5. REVIEW 상태 사유

촬영 품질은 PASS이지만, 상담 메모에 트레이너가 직접 확인해야 할 항목이 포함되어 REVIEW 상태로 분류합니다.

| 확인 필요 항목 | 설명 |
|---|---|
| 주관적 불편감 메모 | 회원이 특정 동작 중 불편감을 언급한 것으로 기록됨 |
| 운동 강도 체감 | 최근 세션의 주관적 강도가 이전보다 높게 기록됨 |
| 운동 지속성 | 최근 출석 간격이 일정하지 않아 상담 확인 필요 |
| 목표 재확인 | 회원의 초기 목표와 현재 체감 목표가 달라졌는지 확인 필요 |

## 6. 트레이너 확인 포인트

| 확인 항목 | 트레이너 확인 방향 |
|---|---|
| 불편감 표현 | 통증 원인을 판단하지 말고, 회원이 말한 내용을 상담 메모로 확인 |
| 운동 강도 | 다음 세션 전 주관적 피로도와 수행 가능 범위 확인 |
| 출석 패턴 | 운동 지속을 어렵게 만든 생활 일정이나 환경 요인 확인 |
| 목표 변화 | 감량, 체형 변화, 컨디션 관리 등 우선순위 재정리 |
| 필요 시 안내 | 지속적 불편감이 있으면 적절한 전문가 상담을 안내 |

## 7. 상담 리포트 예시 문장

이번 촬영은 카메라, 조명, 구도, 배경, 바닥 기준 마커 조건이 모두 PASS로 확인되었습니다. 따라서 촬영 데이터는 상담 리포트 작성에 사용할 수 있는 기본 품질 조건을 충족한 것으로 볼 수 있습니다.

다만 상담 메모에 특정 동작 중 불편감 표현과 최근 운동 강도 체감 상승이 기록되어 있어, 자동 리포트 확정 전 트레이너 확인이 필요합니다.

본 리포트는 운동상담 참고 자료이며, 의료적 진단이나 치료 판단을 제공하지 않습니다.

## 8. 주의 문구

- 본 리포트는 합성 데이터 기반 샘플입니다.
- 실제 회원 이미지나 개인정보를 포함하지 않습니다.
- 의료 진단, 치료, 재활, 처방 목적이 아닙니다.
- 불편감 표현은 진단 근거가 아니라 상담 확인 항목입니다.
- 최종 상담 내용은 트레이너가 직접 확인해야 합니다.

## 9. 사업계획서 활용 문장

LIGHT ONE은 촬영 품질이 기준을 충족하더라도 상담 내용상 트레이너 확인이 필요한 경우 REVIEW 상태로 분류할 수 있습니다. Review 샘플 리포트는 자동 판단을 확정값으로 제공하는 것이 아니라, 트레이너가 확인해야 할 상담 항목을 구조화하는 Human-in-the-loop 흐름을 보여줍니다.

본 샘플은 실제 회원 데이터가 아닌 합성 데이터 기반이며, 서비스의 상담 보조 흐름을 설명하기 위한 목적으로 작성되었습니다.
