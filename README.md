# LIGHT ONE Healthcare AI Platform

LIGHT ONE은 체형 분석, 허리 통증 예측, AI 의사결정 안전장치(CADSS)를 하나의 헬스케어 AI 사업 흐름으로 통합하기 위한 프로젝트 저장소입니다.

## Core Themes

- Posture and body-shape analysis
- Low back pain risk prediction and care support
- CADSS: Controlled AI Decision Support System
- Healthcare AI governance, safety, and explainability

## Current Scope

This repository will host the LIGHT ONE healthcare AI website and CADSS technical landing page assets.
