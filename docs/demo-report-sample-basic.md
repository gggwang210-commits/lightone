# LIGHT ONE 데모 상담 리포트 샘플 Basic

## 1. 문서 목적

이 문서는 LIGHT ONE 서비스가 실제 PT 현장에서 어떤 형태의 상담 리포트로 활용될 수 있는지 보여주기 위한 합성 데이터 기반 샘플입니다.

이 문서에는 실제 회원 이미지, 얼굴 이미지, 실명, 연락처, 건강정보, 상담 원본 데이터가 포함되어 있지 않습니다. 모든 내용은 포트폴리오와 심사 설명을 위한 예시입니다.

## 2. 리포트 위치

LIGHT ONE은 현재 완성된 상용 서비스가 아니라, PT 상담 리포트의 신뢰도를 높이기 위한 촬영 품질 확인, 조명 품질 측정, 카메라 보정, 촬영 부스 QC 기술 증빙 MVP입니다.

본 리포트는 의료 진단, 치료 판단, 재활 처방, 질환 위험도 예측을 제공하지 않습니다. 트레이너가 회원 상담 시 참고할 수 있는 운동상담 보조 자료입니다.

| 상담 목적 | 체형 변화 기록 및 운동 상담 |
| 주요 목표 | 자세 관찰, 운동 습관 기록, 상담 리포트 생성 |
| 데이터 유형 | 합성 데이터 |

## 4. 촬영 품질 확인 결과

| 항목 | 결과 | 설명 |
|---|---|---|
| 카메라 캘리브레이션 | PASS | 합성 체커보드 기반 재투영 오차가 기준 범위 내에 있음 |
| 조명 균일도 | PASS | 합성 조명 패치 기준 균일도가 기준 범위 내에 있음 |
| 그림자 비율 | PASS | 그림자 영향이 낮은 상태로 계산됨 |
| 피험자 구도 | PASS | 프레임 중앙 배치와 전신 포함 조건이 충족됨 |
| 배경 단색 통제 | PASS | 배경 편차가 기준 범위 내에 있음 |
| 바닥 기준 마커 | PASS | 거리/스케일 확인용 기준 마커가 검출됨 |

## 5. 상담 요약

이번 샘플 리포트에서는 촬영 조건이 PASS로 확인되어 상담 리포트에 사용할 수 있는 입력 품질 조건을 충족한 것으로 표시됩니다.

트레이너는 이 결과를 바탕으로 회원의 운동 기록, 상담 메모, 주관적 피로도, 통증 호소 여부, 촬영 기록을 함께 확인할 수 있습니다.

단, 본 결과는 체형 상태를 확정하거나 통증 원인을 판단하는 자료가 아닙니다. 상담 참고용으로만 사용해야 합니다.

## 6. 트레이너 확인 포인트

| 확인 항목 | 설명 |
|---|---|
| 촬영 조건 | 카메라, 조명, 구도, 배경, 바닥 마커가 기준을 만족하는지 확인 |
| 운동 기록 | 최근 세션의 운동 수행량과 주관적 강도를 확인 |
| 상담 메모 | 회원이 직접 말한 불편감, 목표, 생활습관 변화를 확인 |
| 재촬영 필요 여부 | QC가 CHECK 또는 FAIL이면 리포트 작성 전 재촬영 검토 |
| 최종 판단 | 자동 결과가 아니라 트레이너가 최종 상담 내용을 확인 |

## 7. 상담 리포트 예시 문장

이번 촬영은 카메라, 조명, 구도, 배경, 바닥 기준 마커 조건이 모두 PASS로 확인되었습니다. 따라서 이번 세션의 촬영 데이터는 상담 리포트 작성에 사용할 수 있는 기본 품질 조건을 충족한 것으로 볼 수 있습니다.

최근 운동 기록과 상담 메모를 함께 확인한 결과, 회원은 운동 수행을 지속하고 있으며 상담 시 목표와 체감 변화를 함께 점검하는 것이 적절합니다.

본 리포트는 운동상담 참고 자료이며, 의료적 진단이나 치료 판단을 제공하지 않습니다.

## 8. 주의 문구

- 본 리포트는 합성 데이터 기반 샘플입니다.
- 실제 회원 이미지나 개인정보를 포함하지 않습니다.
- 의료 진단, 치료, 재활, 처방 목적이 아닙니다.
- 통증이나 불편감이 지속되는 경우 적절한 전문가 상담이 필요할 수 있습니다.
- 최종 상담 내용은 트레이너가 직접 확인해야 합니다.

## 9. 사업계획서 활용 문장

LIGHT ONE은 PT 현장에서 상담 리포트를 생성하기 전, 촬영 데이터가 비교 가능한 조건인지 확인하고 PASS/CHECK/FAIL로 분류합니다. Basic 샘플 리포트는 촬영 조건이 적절한 경우 트레이너가 어떤 방식으로 상담 자료를 확인할 수 있는지 보여주는 예시입니다.

본 샘플은 실제 회원 데이터가 아닌 합성 데이터 기반이며, 서비스의 출력 형태와 상담 보조 흐름을 설명하기 위한 목적으로 작성되었습니다.
