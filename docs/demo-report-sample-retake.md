# LIGHT ONE 데모 상담 리포트 샘플 Retake

## 1. 문서 목적

이 문서는 LIGHT ONE 서비스에서 촬영 조건이 기준을 충족하지 못해 재촬영이 필요한 RETAKE 상태를 보여주기 위한 합성 데이터 기반 샘플입니다.

이 문서에는 실제 회원 이미지, 얼굴 이미지, 실명, 연락처, 건강정보, 상담 원본 데이터가 포함되어 있지 않습니다. 모든 내용은 포트폴리오와 심사 설명을 위한 예시입니다.

## 2. 리포트 위치

LIGHT ONE은 PT·피트니스 현장에서 상담 리포트의 신뢰도를 보조하기 위한 비의료 운동상담 보조 SaaS를 목표로 합니다.

본 리포트는 의료 진단, 치료 판단, 재활 처방, 질환 위험도 예측을 제공하지 않습니다. 촬영 조건이 기준을 벗어난 경우, 트레이너가 리포트 확정 전에 재촬영 여부를 확인할 수 있도록 돕는 참고 자료입니다.

| 항목 | 결과 | 설명 |
|---|---|---|

| 바닥 기준 마커 | FAIL | 거리/스케일 기준 마커가 충분히 검출되지 않음 |

## 5. RETAKE 상태 사유

이번 샘플은 촬영 조건 중 피험자 구도와 바닥 기준 마커 항목이 기준을 충족하지 못해 RETAKE 상태로 분류합니다.

| 실패 항목 | 재촬영 사유 |
|---|---|
| 피험자 구도 | 피험자가 프레임 중앙에서 벗어나 전후 비교 신뢰도가 낮아질 수 있음 |
| 바닥 기준 마커 | 거리/스케일 기준점이 충분히 확인되지 않아 크기 비교가 어려울 수 있음 |
| 조명 상태 | 일부 그림자 또는 밝기 편차가 있어 촬영 환경 재확인 필요 |

## 6. 트레이너 안내 문구

이번 촬영은 일부 품질 조건이 기준을 충족하지 못했습니다. 상담 리포트에 사용하기 전, 아래 조건을 확인한 뒤 다시 촬영하는 것을 권장합니다.

- 회원이 바닥 중앙 마커 위에 서 있는지 확인
- 전신이 프레임 안에 충분히 들어오는지 확인
- 바닥 기준 마커가 화면 하단에 선명하게 보이는지 확인
- 조명이 한쪽으로 치우치지 않았는지 확인
- 배경에 무늬나 그림자가 강하게 생기지 않았는지 확인

## 7. 상담 리포트 예시 문장

이번 촬영은 카메라 보정과 배경 조건은 기준을 충족했으나, 피험자 구도와 바닥 기준 마커 검출 항목에서 기준 미달이 확인되었습니다.

따라서 이번 촬영 데이터는 상담 리포트에 바로 사용하기보다 재촬영 후 다시 품질 확인을 진행하는 것이 적절합니다.

본 리포트는 운동상담 참고 자료이며, 의료적 진단이나 치료 판단을 제공하지 않습니다.

## 8. 주의 문구

- 본 리포트는 합성 데이터 기반 샘플입니다.
- 실제 회원 이미지나 개인정보를 포함하지 않습니다.
- 의료 진단, 치료, 재활, 처방 목적이 아닙니다.
- RETAKE는 건강 상태 판단이 아니라 촬영 품질 기준 미달을 의미합니다.
- 최종 상담 내용은 트레이너가 직접 확인해야 합니다.

## 9. 사업계획서 활용 문장

LIGHT ONE은 촬영 조건이 기준을 충족하지 못한 경우 결과를 확정하지 않고 재촬영 사유를 안내할 수 있습니다. Retake 샘플 리포트는 촬영 품질이 낮은 입력을 그대로 상담 리포트에 사용하지 않고, 구도·조명·바닥 기준 마커 등 재현성 조건을 먼저 확인하는 품질 관리 흐름을 보여줍니다.

본 샘플은 실제 회원 데이터가 아닌 합성 데이터 기반이며, 서비스의 촬영 품질 관리 흐름을 설명하기 위한 목적으로 작성되었습니다.
